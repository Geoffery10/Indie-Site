## About the Site

This site is a creative space for:
- **Digital Art & Photography**: Showcasing visual works.
- **Projects**: Documenting software and hardware experiments.
- **Blog**: Sharing thoughts and updates on the indie web.

### Features
- Retro-inspired aesthetic with pixel typography.
- Daily Bible Verse integration.
- Responsive design with a unique green fluid background.
- Navigation for Art, Blogs, and Projects.
- **Powered by Indie API**:
    - **Articles**: Dynamic serving of markdown-based projects and blogs.
    - **Art**: Integration with Immich for asset management.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS3, JavaScript.
- **Backend**: [Indie API](https://github.com/Geoffery10/Indie-API) (ASP.NET Core).
- **Typography**: VT323 (Google Fonts).
