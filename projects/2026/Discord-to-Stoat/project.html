Discord to Stoat Migration


With all the recent drama around Discord's age verification I decided it was time to start preparing for the worst. Discord is a platform I've enjoyed using for the past 10 years. Sadly the platform has continued to become less free and private and more ad-filled and monitored. While I don't think Discords new age verification will be the nail in the coffin I've seen the writing on the wall for some time now.




That's where Stoat (formerly known as "Revolt") comes in. I considered many options before choosing Stoat and still I consider Matrix as another great option. I chose Stoat because it was the most Discord-like of the self-hosted alternatives out there. With it I was able to migrate all my Discord channels, messages, and media. To do this effectively I used Discord Chat Exporter to export all the old messages and media from Discord. Then I wrote a custom Python Stoat Bot I called "Discord Restore" which would read the backed up messages and media and post them one by one to the new Stoat channels. This way I was able to preserve the original timestamps and authorship of all the old messages.




The migration process was pretty smooth overall. The Discord Restore bot worked well enough with only a few posts (mostly embeds) failing to be migrated properly. The result was a near perfect replica of the old Discord server on my new Stoat instance hosted securely on my own server.




Unfortunately, it's not all sunshine and rainbows. Stoat has a major issue... it's voice and video chat features are currently non-functional in the self-hosted version. This is a huge problem as my friends and I use voice chat daily. This is by far the biggest reason we can not switch to Stoat full time.



Thankfully, Discord is still an option for now so I don't have to rush having a feature complete alternative just yet. In the meantime I'll be keeping an eye on Stoat's development and watching other alternatives like Matrix as well. Even if we never move off Discord completely, it's nice to have a secure self-hosted backup of all our old messages and media just in case.
