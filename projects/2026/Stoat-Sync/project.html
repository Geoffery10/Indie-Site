Stoat Sync


Since I started the move to Stoat which you can read about here there was one important thing I was missing. The backups from Discord to Stoat got the history of each channel in place but any new messages would have to be backed up again and manually synced. This would not be sustainable long-term. That's why I knew I needed to build Stoat Sync.

Stoat Sync is a real-time synchronization bridge between Discord and Stoat. It monitors specified Discord channels and mirrors all activity to corresponding Stoat channels, including new messages, edits, and deletions. This insures that my Stoat instance remains a perfect mirror of my Discord and vice versa.

The project has been a fun challenge to build and involved learning Stoats vague and mostly undocumented API. I also had to figure out how to make both a Discord and Stoat bot run simultaneously. This project also gave me a chance to brush up on my TDD skills although due to the pace of development TDD was more of an "after the fact" instead of the preferred "test first" approach.


Overall I'm really happy with how Stoat Sync turned out. It still has some bugs and edge cases like lack of emoji translation, lack of embed support, and issues where Discord has more attachments than Stoat's max of 5. I hope to continue polishing this project more and I hope others may find it as useful as I have.
